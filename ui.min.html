Select

Select an account to copy its password to the clipboard. The clipboard will be cleared automatically after 30 seconds.

Create

Enter and save an account label and password. Account labels can include usernames or hints. You can use the Generate Strong Password button to create a high-entropy, 144-bit password.

If a website rejects a generated password, simply create another and manually edit it to meet the site's specific requirements before saving.

Relabel

Select an account and enter a new label. This only changes the label, not the password.

Delete

Select an account to delete it. You will be prompted with a confirmation box; once deleted, the password cannot be recovered.

Security

Passwords are encrypted using Windows Data Protection and stored in a local file. They can only be decrypted on your computer by someone logged in as you.

Backup

The passwords file is in your AppData folder. C:\Users\<username>\AppData\Local
 \Programs\Keyvok\passwords.db
The AppData folder is hidden by default.

A backup of this file is only usable on this computer by someone logged in as you. It will not work after reinstalling Windows or replacing the motherboard.

To keep a portable copy of your passwords, export them as a CSV file, ideally to a flash drive kept in a safe place.
